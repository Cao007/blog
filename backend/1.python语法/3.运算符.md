## 赋值运算符
1.=

~~~python
# a、b、c分别赋值
a,b,c=1,2,3

# d、e赋相同的值
d=e=4
~~~

2.复合赋值运算符

~~~python
a = 1

# a b c都赋值为1
a = b = c = 1

# a b c分别赋值为1 2 3 
a,b,c = 1,2,3

# 跳过赋值 
a,_ = 1,2
~~~
